# testNisum
Test springboot jcruz.

# Solución de prueba tecnica.

## Tecnología utilizada

- Java jdk 1.8
- SpringBoot 2.6.3
- h2 Database
- jwt
- spring openapi 1.6.4
- maven

## Clonar el proyecto:
git clone https://github.com/jaime2496333/testNisum.git

## Ejectuar el proyecto
mvn spring-boot:run
